## Summary

Implements #131 — first-class non-interactive mode for `create-cf-token` with scope discovery, declarative token specs, agent skill output, and programmatic library exports.

- **Discovery**: `--list-scopes`, `--list-permissions`, `--list-accounts` with JSON/table output (no token created)
- **Non-interactive create**: `-n`, `--name`, `--preset full-access`, `--accounts`, `--scopes`, `--output json`, `--dry-run`, `create --file`
- **Agent guidance**: `--skill` (and `--help skill` alias) prints full playbook from `assets/automation/`; `--help automation` for flag reference
- **Repo mirror**: `skill/create-cf-token/` synced via `bun run sync-skill`
- **Library**: `create-cf-token/create`, `/spec`, `/scope-spec`, `/policies` sub-exports
- **Non-TTY safety**: incomplete specs fail fast with actionable errors instead of hanging
